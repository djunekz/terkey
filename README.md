# TERKEY Termux Keypad
TERKEY digunakan untuk menambah tombol PGUP PGDN KANAN KIRI dan lainnya jadi lebih mudah dalam penggunaan termux
# install

**Install**

```
$apt install python git -y
$git clone https://github.com/djunekz/terkey
$cd terkey
$python terkey.py
```
**Single Install**

```
apt update && pkg install python git && git clone https://github.com/djunekz/terkey && cd terkey && python terkey.py
```
